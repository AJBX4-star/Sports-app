<original_problem_statement>
Build me a sports squares app that is a 10 by 10 square grid. The titles for the horizontal and vertical axis need to be customizable. I need to be able to local host games where each player can see the game and select squares in order until all 100 squares are claimed. At a time set by the user the numbers 0-9 will automatically randomize across the top and down the left side of the grid. Winning square after each quarter will be selected by the host and will highlight green for everyone in the hosts party.

PRODUCT REQUIREMENTS (based on subsequent requests):
- Add numbers 1-100 on the squares.
- Host can set the number of picks allowed per person per turn.
- Host can choose player draft order (Standard or Snake).
- Host can claim a square for another player.
- Add sports-themed imagery to the main page (user-provided).
- Lock squares once selected.
- Host can claim a square for unclaimed.
- Board locks when all squares are full; randomization is only allowed after the board is locked.
- Undo functionality for accidental claims.
- Host can manually add/remove players.
- Track and display live scores.
- Team A name on top, Team B name on the left side.
- Host claims for others should count towards that player's turn picks.
- Highlight the current winning square based on live score tracking.
</original_problem_statement>
